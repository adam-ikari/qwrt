Why qwrt

A runtime designed from the ground up for embedding — no compromises.

⚡

Strict C99

Embeddable in any C99 codebase. No C11 requirement on the host; C11 deps (QuickJS-ng, libuv) are isolated to their own compilation units.

📦

Zero System Libs

QuickJS-ng, mbedTLS, miniz, libuv, wasm3 — all built from source via add_subdirectory. Fully self-contained, no system package dependencies.

🌐

WinterCG Polyfill

fetch, console, crypto.subtle, ReadableStream, timers, fs, URL, TextEncoder — 21 modules, precompiled to bytecode for instant loading.

🔌

Portable PAL

Same JS code, three platforms: libuv (Linux/macOS), FreeRTOS (ESP32-S3), mock (tests). Add your own backend with ~30 function pointers.

🧵

Multi-Context

Spawn, suspend, and resume isolated JS contexts within one runtime. Each context has its own PAL and permissions — sandbox by default.

🔒

No Globals

Zero mutable file-scope state. Per-runtime isolation via qwrt_t — safe to run multiple independent instances in one process.

Quick Start

Clone, build, and run — three commands.

Shell 
# Clone with all submodules
git clone --recursive https://github.com/adam-ikari/qwrt.git
cd qwrt

# Configure & build
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)

Minimal Example

Create a runtime, evaluate JS, drive the event loop — all in ~20 lines of C.

main.c 
#include <qwrt/qwrt.h>
#include <pal_uv.h>

int main(void) {
    qwrt_pal_t *pal = pal_uv_create(NULL);
    qwrt_t *rt = qwrt_create(&(qwrt_config_t){ .pal = pal });

    qwrt_eval(rt, "console.log('Hello from QuickJS!');", NULL);

    char *result = NULL;
    qwrt_eval(rt, "1 + 1", &result);
    printf("1 + 1 = %s\n", result);  /* "2" */
    qwrt_free(result);

    while (pal->run_cycle(pal, 100) > 0) qwrt_tick(rt);

    qwrt_destroy(rt);
    return 0;
}

JS Polyfill Modules

21 WinterCG-compatible modules, precompiled to bytecode, injected at runtime.

🌐

fetch

fetch(), Headers, Request, Response — streaming HTTP with abort support.

🖨️

console

console.log, warn, error, debug, info — formatted output via PAL log.

🔐

crypto

crypto.subtle — SHA-256/384/512, HMAC, PBKDF2, AES-GCM via mbedTLS.

🌊

streams

ReadableStream, WritableStream — WHATWG Streams API for incremental I/O.

⏱️

timers

setTimeout, setInterval, clearTimeout, clearInterval — backed by PAL timers.

📁

fs

fs.read, fs.write, fs.exists, fs.remove, fs.list — async filesystem access.

💾

storage

storage.get, storage.set, storage.delete — key-value persistence.

🔤

encoding

TextEncoder, TextDecoder — UTF-8 and Base64 encoding/decoding.

🔗

url

URL, URLSearchParams — WHATWG URL parsing and query string handling.

🛑

abort

AbortController, AbortSignal — cancel async operations (fetch, streams).

📊

performance

performance.now() — high-resolution monotonic time via PAL hrtime.

📡

event-target

EventTarget, Event, CustomEvent — DOM-style event dispatch.

💬

message-channel

📦

🧭

🔄
